Formulario Fullstack Developer

Este proyecto comenzó como un formulario básico en React y fue evolucionando hasta convertirse en una aplicación más profesional, con mejores prácticas, validaciones modernas y una interfaz más cuidada.

¿Qué hice en este proyecto?

Decidí mejorar un formulario simple para llevarlo a un nivel más cercano a lo que se usa en entornos reales de trabajo.

Los principales cambios fueron:

Mejoré la interfaz usando Tailwind CSS

Separé el código en componentes reutilizables

Implementé validaciones con React Hook Form y Zod

Organicé mejor la estructura del proyecto

Pensé el formulario desde un enfoque real (encuesta para desarrolladores)


Validación de campos en tiempo real

Selección múltiple (checkbox).
Selección única (radio).
Formularios controlados de manera eficiente.
Reset automático al enviar.

El objetivo principal fue practicar:

Buenas prácticas en frontend.
Componentización.
Validación de formularios.
Mejora de UI/UX.

Reflexión:

Este proyecto me ayudó a entender cómo pasar de algo básico a algo más profesional, pensando no solo en que funcione, sino en que sea mantenible, escalable y más cercano a un producto real.
